About the Project

The Employee Management System is a full-stack web application created to replace manual employee data handling with a structured digital process.

Why this project

Traditional paper files and spreadsheets are difficult to maintain, insecure, and time-consuming. This application gives Admin or HR users a centralized platform to manage records efficiently.

Technologies used

Frontend: HTML and CSS with responsive design
Backend: Flask using Python
Database: MySQL with hashed passwords
Security: Session management and Werkzeug password hashing

Key Features

  • Secure Admin Login with password hashing
  • Add, View, Edit, Delete employee records (CRUD)
  • Upload employee photos
  • Search employees by name or department
  • Dashboard with employee statistics
  • Template inheritance for code reusability

Project Workflow

  • Admin Registration → Create account
  • Admin Login → Secure authentication
  • Dashboard → View employee stats
  • Add Employee → Enter new records
  • View Employees → See all employees
  • Edit/Delete → Update or remove records
  • Logout → End session

Real-Time Application

This system follows real-time organizational practices where employee data is controlled by HR/Admin, not by employees, to ensure security and data integrity.

Future enhancements can include role-based access, payroll modules, attendance tracking, and email notifications.
