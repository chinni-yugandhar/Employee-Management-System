This Employee Management System is a student project developed using Flask, HTML, CSS, and MySQL. It demonstrates secure authentication, CRUD operations, and file upload functionality.

Project Details

Database: MySQL
Frontend: HTML5 & CSS3
Features: Password Hashing, Photo Upload, Search Functionality
